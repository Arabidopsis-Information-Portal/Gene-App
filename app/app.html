This app displays gene data (in table format) obtained from ThaleMine , an Arabidopsis-specific instance of InterMine . InterMine is an open source data warehouse built specifically for the integration and analysis of biological data. The specific data displayed in the app includes gene annotation (TAIR & JCVI), GO (Gene ontology) annotation, GeneRIFs (NCBI), and publications associated with a gene (UniProt & NCBI).

Instructions

To get started, enter an AGI locus identifier in the search box and click on the Search button. At this point, data will be loaded in four tabs labeled Summary, GO, GeneRIFs, and Publications. The number of records found in each category will be displayed on the tab. The Summary tab displays summary gene annotation information such as gene ID, name, symbol, computational description, curator summary, and location data. The GO tab lists all of the Gene ontology terms associated with the gene. The GeneRIFs tab lists all of the associated additions to the functional annotation of the gene. Finally, the Publications tab lists all of the publications associated with the selected gene.

Features (Tables)

  • Sort the table by any column by clicking on the column header.
  • Change the columns that are shown by clicking on the Column visibility button and then selecting/de-selecting the columns.
  • Export all of the data in the GO, GeneRIFs, or Publications table in CSV or Excel format by clicking on the CSV or Excel button.
  • Export options:
    • When exporting CSV or Excel data, the file is saved without prompting in the browser's current download location and will be named as follows:
      • For GO data:
        • GO_terms_for-<query>.(csv/xlsx)
      • For GeneRIF data:
        • GeneRIFs_for-<query>.(csv/xlsx)
      • For Publication data:
        • Publications_for-<query>.(csv/xlsx)
    • NOTE: when using Safari for the Mac
      • Excel export may not work (Adobe Flash is required)
      • The file will be opened in a new tab or window and File->Save As... must be used to save the file.
  • Filter the results further by entering text in the Search box in the upper-right of the table.
  • Click the expand icon next to the PubMed ID in the GeneRIFs table to show detailed information about the publication.
  • Clicking on a PubMed ID will display the publication at NCBI.

The AGI locus identifier follows the format: ATXgYYYYY. The 'AT' refers to Arabidopsis thaliana, the 'X' number corresponds to the chromosome number, the 'g' refers to gene, and the 5 following 'Y(s)' correspond to the five-digit code, numbered from top to bottom of the chromosome.
